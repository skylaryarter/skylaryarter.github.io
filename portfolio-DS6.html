The Making of Wuggy the Productivity Pet: Design in Practice

Project Date May 13th, 2025
Read More Full write-up

For our final design sprint, we were tasked with following the iterative design process in order to create a browser extension from scratch. This involved ideation, design, user testing, and iterating on the design based on user feedback.

After brainstorming several ideas, we decided to design a productivity pet that grows with every task the user completes. We used HTML, CSS, and JavaScript to code our extension, and then conducted user testing to assess its usability and improve the design.
